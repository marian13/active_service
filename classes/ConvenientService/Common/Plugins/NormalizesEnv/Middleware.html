Class ConvenientService::Common::Plugins::NormalizesEnv::Middleware < Core::ClassicMiddleware

  • lib/convenient_service/common/plugins/normalizes_env/middleware.rb

  • Single splat ‘*’ converts ‘nil’ to empty array.

  • Double splat ‘**’ raises on ‘nil’.

  • Umpersand ‘&’ converts ‘nil’ to ‘nil’.

The following decorator ‘proc’ converts ‘env’ to a hash. This way ‘stack.call(*env, **env, &env)’ won’t fail even if a user passes ‘nil’ as ‘kwargs’.

def call(env = nil)
  env = env.to_h
  env = env.merge(args: env[:args].to_a, kwargs: env[:kwargs].to_h, block: env[:block])

  stack.call(env)
end
